# AXON Production Documentation

Enterprise architecture and production-readiness planning for AXON. These documents describe **future hardening work** — not current shipped capabilities.

| Document | Audience | Purpose |
|----------|----------|---------|
| [ENTERPRISE_PRODUCTION_PATH.md](ENTERPRISE_PRODUCTION_PATH.md) | Platform engineers, SRE, security, product operations, robotics ops leadership | Staged enterprise hardening roadmap, readiness matrix, deployment options, risks, and evidence links |

**Scope boundary:** AXON today is a simulated-only, synthetic-only, local-first prototype. It is not enterprise-production-ready. AXON is not a medical device, not a diagnostic system, not for clinical use, and does not diagnose or treat any condition. Production documents describe investment paths and guardrails — not claims that production deployment is available or low-cost today.
