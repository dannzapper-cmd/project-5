Live Demo Cockpit

This is a live, locally-running operations cockpit — not static HTML. The list below explains what to watch; every button logs an action and shows a visible result. Optional modules state their activation command instead of failing silently.

  • Live backend / API / WebSocket connectivity (Backend Proof panel + status dots).
  • Synthetic telemetry streams (EMG, ECG-like, IMU, SpO2-proxy) updating in real time.
  • Edge inference / model score updates with per-event latency.
  • Safety + Human-in-the-Loop decisions you can confirm or reject.
  • Digital twin state changes after operator commands.
  • Evidence-backed optional modules (FL / RL / MLOps) and clear profile boundaries.
